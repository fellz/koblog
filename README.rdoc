== Project info

Collective blog with following features:

* Authorization with devise

* Authorized users can post articles

* Post moderation

* Admin panel for approval/rejecting posts

* Email notifications

* Category subscribtions with email notifications about new posts


Stack: 

* Ruby 2.2.4

* Rails 4.2.6

* Frontend: jQuery, Coffeescript

* Database PostgreSQL

* Redis for Sidekiq

* Sidekiq for email sending

* Services - Elasticsearch

* Deployment - heroku 


Made within 10 days.
